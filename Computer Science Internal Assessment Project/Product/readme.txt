
Installation:
_______________________________________________________________________________________________________

In order to run the program the user must be in the product directory.


This application requires mySQL to be installed in order to operate.
_______________________________________________________________________________________________________

In order to install database:


Type in terminal/ command prompt: javac DBInstallation.java

Then type in terminal/ command prompt: java DBInstallation

_______________________________________________________________________________________________________

The program can be run in two ways: by jar file, or by command in terminal/ command prompt.


To run by jar:

Type in terminal/ command prompt: java -jar Alumni.jar


To run by command in terminal/ command prompt:

For Mac, type in terminal: java -cp ../mysql-connector-java-5.1.40-bin.jar:. AlumniDatabase

For Windows, type in command prompt: java -cp ../mysql-connector-java-5.1.40-bin.jar;. AlumniDatabase
